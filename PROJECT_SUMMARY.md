# Financial Sentiment Analysis Project

## Цель проекта
Анализ тональности финансовых новостей с использованием машинного обучения.

## Данные
- Источник: Financial PhraseBank
- Размер датасета: 4846 записей
- Распределение:
  - Neutral: 2879 (59.4%)
  - Positive: 1363 (28.1%)
  - Negative: 604 (12.5%)

## Модель
- Алгоритм: Logistic Regression
- Векторизация: TF-IDF
- Точность: ~85-90%

## Использование
1. Активировать виртуальное окружение: `source venv/bin/activate`
2. Запустить демо: `python src/demo.py`
3. Ввести текст для анализа

## Результаты
Модель успешно определяет тональность финансовых новостей.
