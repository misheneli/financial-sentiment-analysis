# Financial Sentiment Analysis Project

Проект для анализа тональности финансовых новостей с использованием машинного обучения.

## 📋 О проекте

Этот проект использует методы обработки естественного языка (NLP) и машинного обучения для классификации тональности финансовых новостей на три категории:
- 📈 Positive (положительная)
- 📉 Negative (отрицательная)
- ➖ Neutral (нейтральная)
